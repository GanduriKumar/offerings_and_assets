## Report Assembly Rules
- start with the recommendation, not the research method
- keep every workflow on one page where possible
- avoid generic AI strategy language
- show why some workflows should be rejected
- name owners and conditions explicitly
- keep the output usable in executive approval discussions

## Potato Mode Validation

### The Pack Works If
- each template leads to a concrete client decision
- weak opportunities can be rejected quickly
- the report can be produced without inventing a large consulting framework
- the outputs create a natural bridge to pilot design or follow-on advisory

### The Pack Fails If
- the report reads like a generic maturity assessment
- there is no visible recommendation category
- conditions to proceed are unclear
- ownership of the next move is missing

### Verdict
This template pack is worth using because it supports the exact promise already made in the offering collateral. It stays commercially useful only if the final reports remain short, explicit, and hard to misread.
