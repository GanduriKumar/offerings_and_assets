## Report Assembly Rules
- start with the recommendation, not the method
- keep the readiness view concise and readable
- make the opportunity shortlist selective
- show where leadership should slow down, not just where it should move
- name owners and conditions explicitly
- avoid generic AI strategy language

## Potato Mode Validation

### The Pack Works If
- leadership can see the readiness picture quickly
- the opportunity shortlist is selective
- the next action is practical and owned
- weak or premature opportunities are clearly deprioritized

### The Pack Fails If
- the report turns into a generic maturity exercise
- there is no visible recommendation
- risks are vague
- the output does not guide an immediate decision

### Verdict
This template pack works if the final report stays short, pointed, and commercially useful for executive decision-making.
